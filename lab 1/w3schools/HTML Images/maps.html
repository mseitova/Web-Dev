Shape Examples

Here are examples of shapes you can define in an image map:

  • Rectangle: Define a rectangular region using coordinates for two corners (top-left and bottom-right).
  • Circle: Define a circular region using the center point and radius.
  • Polygon: Define a polygonal region by specifying multiple points.
